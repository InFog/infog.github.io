Olá!

Bom, hoje eu vou fazer um anúncio de mais um software que estou desenvolvendo. O software em questão é o JSCalc.


O JSCalc é uma calculadora feita em javascript, ela é bem simples, faz as operações básicas. A grande sacada da JSCalc é a exibição dos digitos parecidos com uma calculadora de verdade.


A JSCalc é um fork de um projeto de um amigo meu, na página da JSCalc no Google Code está melhor explicado como e porque eu fiz o fork.


Se você está estudando ou quer aprender javascript e um pouco de css eu recomendo que baixe a JSCalc, o código está bem comentado e simples de entender.
