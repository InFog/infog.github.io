Acabei de colocar os pacotes do GeSpeak versão 0.4 beta 2 para download no Google Code.

Nesta versão não precisamos mais escolher os idiomas 'en', 'pt', 'es', agora está tudo mais amigável! Os nomes dos idiomas agora são 'english', 'brazil', 'spanish', etc, Tudo bem que estão em inglês, mas acredito que assim já é bem mais fácil para escolher. Ahh, quem pediu esta funcionalidade foi o Kretcheu. Valeu, Kretcheu!

Hello people!

I have just added GeSpeak version 0.4 beta 2 packages for download at Google Code!

In this version we don't need to choose languages like 'en', 'pt', 'es' anymore now everything is user friendly! The languages' names are now 'english', 'brazil', 'spanish', etc. Kretcheu was the one who asked for this. Thanks, Kretcheu!

Another new feature is to write output to a WAV file, this one was asked by Kalib. There it is, Kalib! Thanks for your suggestion =)

And you can also help GeSpeak to grow! How to do it? By suggesting new features, programming or translating. Actually GeSpeak is translated into English and Brazilian Portuguese. To translate GeSpeak into new languages download this file and use poedit (or any other text editor) to translate, after this just let me know.

Well, that's it. If there are no problems or bugs reported until next weekend, then I release the final version.
