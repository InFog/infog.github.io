Ah, sim, eu esqueci de falar como abrir arquivos com o Vim, na verdade há duas maneiras mais comuns, uma delas é chamar o Vim passando o nome do arquivo como argumento:

$ vim arquivo.txt

Outra forma é dentro do vim usar o comando ":open arquivo.txt". Para usar os comandos do Vim você precisa primeiro usar a tecla <esc>... Hum, isso está ficando meio complicado de explicar com um texto, acho que vou mostrar com um vídeo, que tal? Fica mais fácil, certo? Ok, aqui vamos nós!
