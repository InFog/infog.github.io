O CakePHP é um ótimo framework para desenvolvimento WEB com PHP e que segue os padões MVC. Se você ainda não conhece um framework ou nunca se interessou em conhecer eu recomendo que aprenda. O engraçado é que no ano passado durante o Yahoo! Hack Day 2008 me perguntaram que framework eu usava e eu disse "Nenhum, ainda não senti necessidade para isso". Mas como eu fui inocente ao dizer esta frase...

Bem, depois disso eu matutei muito e para o meu novo projeto resolvi encarar um framework Eu o estou utilizando há alguns meses, mas só peguei para valer mesmo durante esta semana e acabei descobrindo que a documentação do site do CakePHP é um pouco vazia, as vezes, e por isso resolvi criar um banco de dicas aqui no blog.
