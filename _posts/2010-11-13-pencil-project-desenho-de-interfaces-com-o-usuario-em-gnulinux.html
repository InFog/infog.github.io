Olá!


Não sei quanto à vocês, mas eu sempre quis achar um bom programa para "rabiscar" interfaces com o usuário (user interfaces). Eu já havia usado o ótimo Inkscape para isso, mas é bem trabalhoso criar os formulários e widgets com ele, o ideal seria criar uma biblioteca de widgets desenhadas para usar e mesmo assim ainda não era bem o que eu queria.


Sabe aqueles desenhos de planejamento das interfaces do Ubuntu? Não? Então veja um:



É bem legal, né? Eu comecei a desenhar interfaces da melhor maneira que eu conheço: Papel e canetas, mas sempre ficam rabiscadas e cheias de riscos para apagar partes que eu não quero mais.


Mas esses dias eu encontrei o software que estava procurando, o Pencil Project.



Este software é uma extensão para o Firefox, mas pode ser usado em modo standalone, para isso é necessário ter o xulrunner instalado.


Aqui no meu Debian eu fiz o seguinte:


aptitude install xulrunner

Esse Pencil Project é bem legal e tem widgets em vários formatos, um deles é o Sketchy GUI, dá uma olhada nos widgets dele:



Eu achei muito bom para apresentar planejamento de interfaces para clientes e também para fazer a documentação de projetos.


Recomendo!
