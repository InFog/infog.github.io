Olá pessoal!

Hoje o assunto é a validação de formulários HTML usando o plugin Validation do para jQuery.

Quem já desenvolveu para web tem grandes chances de já ter encarado esse abacaxi. Não que validar formulários usando javascript seja chato, mas também não é bem a parte mais divertida do processo de desenvolvimento de um projeto web.

Para agilizar este processo vou mostrar hoje como utilizar o plugin Validation para jQuery. Com a utilização desta dupla a validação de formulários vai ficar molezinha, e o melhor de tudo é que não é necessário fazer alterações nos seus formulários (Bem, talvez você queira tirar aquelas validações sem graça que exibem "alerts" do javascript, foi o que eu fiz).

Então? Ficou interessado? Então relaxe, prepare seu editor de textos e seu navegador e não tire os olhos deste post até o final =)

O primeiro passo é ter a biblioteca jQuery e também o plugin Validation devidamente adicionados no cabeçalho da página, o código mais abaixo mostra como fazer isso.
